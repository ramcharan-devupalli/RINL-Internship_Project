# RINL Contractor Wage Management System

Internship project for managing contractors and daily wage workers.

## Features Implemented

- User Signup
- Email OTP Verification
- Login Authentication
- JWT-based Sessions
- PostgreSQL Database
- Admin Dashboard
- Contractor Dashboard
- Worker Dashboard

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL

Authentication:
- Nodemailer
- bcrypt
- JWT

## Setup

1. Clone repository
2. Install dependencies

npm install

3. Configure .env

4. Start backend

npm run dev
